# Overview

This WebSharper™ Extension provides a set of classes almost identical to the ones
documented in the [Bing Maps API][bing-api]. When used in WebSharper™ projects, these
stub classes delegate the work to the actual classes implemented in Bing Maps API.

After adding the reference to the project all the classes can be found under the
`IntelliFactory.WebSharper.Bing.Maps` module.

[bing-api]: http://code.google.com/apis/maps/
